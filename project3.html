Plateforme d'échanges

Application web complète d'échanges entre utilisateurs comprenant des un système de question/réponses avec des votes sur les réponses

Client : Demande la construction complète du site en partant de zéro
Date : Septembre 2025 - Janvier 2026
Catégorie : Développement Web Full-Stack
Équipe : Projet en équipe de 5 développeurs
Compétences BUT :
C1 : Réaliser C2 : Optimiser des applications C2 : Administrer des systèmes informatiques communiquant compexes C4 : Gérer des données C5 : Conduire un projet C6 : Collaborer

Liens avec les Compétences du BUT

Ce projet complet m'a permis de développer des compétences fondamentales du BUT Informatique, particulièrement pertinentes pour le parcours RACDV .

C1 : Réaliser un développement d'application

Niveau 3 atteint : Adapter des applications sur un ensemble de supports

  • Développement Full-Stack : Création complète frontend (React) et backend (Node.js/Express)
  • Responsive Design : Application optimisée pour desktop, tablette et mobile
  • Architecture modulaire : Séparation claire des composants (UI, logique métier, services)

C4 : Gérer des données de l'information

Niveau 2 atteint : Optimiser une base de données et mettre en œuvre la sécurité

  • Modélisation de données : Conception de schéma MongoDB pour produits, utilisateurs, commandes
  • Optimisation des requêtes : Indexation des collections fréquemment interrogées
  • Sécurité des données : Chiffrement des informations sensibles (mots de passe, données bancaires)

C5 : Conduire un projet

Niveau 2 atteint : Appliquer une démarche de suivi de projet

  • Méthodologie Agile : Utilisation de Scrum avec sprints de 2 semaines
  • Gestion des exigences : Rédaction des user stories et critères d'acceptation
  • Planification : Estimation des tâches et suivi du burndown chart
  • Livraison continue : Pipeline CI/CD avec déploiement automatique

C6 : Collaborer au sein d'une équipe informatique

Niveau 2 atteint : Situer son rôle et ses missions au sein d'une équipe

  • Collaboration Git : Gestion des branches, résolution de conflits, code reviews
  • Communication technique : Documentation claire et revues techniques régulières
  • Répartition des tâches : Spécialisation frontend/backend avec coordination efficace
  • Rétrospectives : Analyse des succès et axes d'amélioration après chaque sprint

Fonctionnalités Principales

Gestion de ça question en temps réel (création, phases, auteurs, votants)

Dashboard Admin

Interface complète de gestion des utilisateurs en temps réel.

Défis Techniques Relevés

Synchronisation Temps Réel

Maintenir la cohérence des données entre le brouillon utilisateur, et les autres utilisateurs du site

  • Implémentation WebSockets pour les mises à jour de stock
  • Système de verrous optimistes pour éviter les conflits
  • Cache Redis pour les données fréquemment consultées

Sécurité des Transactions

Garantir la sécurité absolue des données clients dans un environnement sensible.

  • Chiffrement AES-256 des données personnelles

Performance à Grande Échelle

Optimiser les performances pour supporter des centaines d'utilisateurs simultanés.

  • Optimisation des requêtes MariaDB avec indexation
