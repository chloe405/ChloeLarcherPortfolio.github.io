Conception et Sécurisation d'une Base de Données MySQL

Modélisation, implémentation et sécurisation complète d'une base de données relationnelle avec respect des normes en vigueur

Période : Janvier - Mai 2025
Catégorie : Base de Données / Sécurité
SGBD : MySQL 8.0
Outil : DBeaver
Compétences BUT :
C4 : Gérer des données C3 : Administrer C1 : Réaliser

Liens avec les Compétences du BUT

Ce projet de conception de base de données m'a permis de développer des compétences clés du BUT Informatique, particulièrement pertinentes pour le parcours C (Administration, gestion et exploitation des données).

C4 : Gérer des données de l'information

Niveau 2 atteint : Optimiser une base de données, interagir avec une application et mettre en œuvre la sécurité

  • Conception complète : Modélisation Merise (MCD, MLD, MPD) d'une base de données relationnelle
  • Optimisation : Indexation stratégique
  • Sécurisation : Implémentation des politiques de sécurité (RGPD)
  • Manipulation : Requêtes complexes (jointures, sous-requêtes, vues, procédures stockées)

C3 : Administrer des systèmes informatiques communicants complexes

Niveau 2 atteint : Déployer des services dans une architecture réseau

  • Installation MySQL : Configuration complète d'un serveur MySQL 8.0 en environnement de production
  • Gestion des utilisateurs : Création et gestion fine des droits d'accès (GRANT, REVOKE)

C1 : Réaliser un développement d'application

Niveau 2 atteint : Partir des exigences et aller jusqu'à une application complète

  • Analyse des besoins : Compréhension et formalisation des exigences métier du client
  • Documentation : Rédaction du cahier des charges technique et de la documentation utilisateur
  • Tests : Validation complète du fonctionnement et des performances
  • Qualité : Respect des standards de codage SQL et des bonnes pratiques

Objectifs du Projet

Création d'une base de données MySQL fonctionnelle, performante et sécurisée répondant aux exigences métier d'un client, avec un focus particulier sur :

Sécurité Maximale

Respect strict des normes de sécurité en vigueur et protection des données sensibles conformément au RGPD.

Performance Optimale

Optimisation des requêtes et de la structure pour garantir des temps de réponse rapides même avec de gros volumes.

Respect du Cahier des Charges

Implémentation exacte des fonctionnalités demandées par le client avec validation à chaque étape.

Maîtrise des Outils

Utilisation experte de DBeaver pour le développement, l'administration et l'optimisation.

Étapes de Conception

Méthodologie Merise

Cycle de développement complet :

  1. Analyse des besoins et recueil des exigences
  2. Modèle Conceptuel de Données (MCD)
  3. Modèle Logique de Données (MLD)
  4. Modèle Physique de Données (MPD)
  5. Implémentation SQL
  6. Tests et optimisation
  7. Documentation et livraison

Technologies et Compétences

Base de Données

  • MySQL 8.0 - SGBD relationnel avec fonctions avancées
  • Moteur MariaDB - Transactions ACID et intégrité référentielle
  • SQL Avancé - Procédures stockées, déclencheurs, vues

Outils

  • DBeaver - IDE complet pour gestion de bases de données
  • MySQL Workbench - Modélisation et administration
  • Git - Versionnement des scripts SQL
  • Draw.io - Création des diagrammes de modélisation

Fonctionnalités Implémentées

Structure de la Base

  • 15 tables normalisées avec relations claires (clés primaires/étrangères)
  • Contraintes d'intégrité référentielle ON DELETE/UPDATE CASCADE
  • Index stratégiques sur les colonnes fréquemment interrogées
  • Vues matérialisées pour les rapports complexes

Gestion des Accès

  • 4 niveaux de privilèges (Admin, Manager, Employé, Lecture seule)
  • Authentification externe possible via PAM ou LDAP
  • Limitation des connexions par compte utilisateur
  • Expiration des mots de passe tous les 90 jours

Exploitation des Données

  • Procédures stockées pour les opérations métier récurrentes
  • Déclencheurs pour la cohérence et l'audit automatique
  • Export automatisé depuis le format CSV/JSON pour reporting
  • Backup incrémental avec compression et chiffrement
